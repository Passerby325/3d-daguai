================================================================================
                        3D RPG 游戏 - 打怪物
                           说明书 v1.0
================================================================================

【游戏简介】
    这是一款基于 Three.js 开发的 3D 第一人称动作游戏。玩家将在一个开放
    的 3D 世界中与 AI 敌人和强大的 Boss 战斗。通过不断击败敌人提升自己
    的能力，挑战越来越难的关卡。

【运行方式】
    1. 确保电脑已安装现代浏览器（Chrome/Edge/Firefox）
    2. 使用本地服务器运行（如 VS Code 的 Live Server 插件）
    3. 在浏览器打开 index.html
    4. 点击"开始游戏"按钮进入游戏

================================================================================
                              操作说明
================================================================================

【基本移动】
    W           前进
    A           向左
    S           后退
    D           向右
    鼠标        控制视角方向
    空格        跳跃
    Shift       加速奔跑（消耗体力）

【战斗操作】
    左键        攻击/射击（当前武器）
    右键        切换武器（剑/手枪）

【其他操作】
    ESC         暂停游戏/解锁鼠标
    M           切换小地图镜像模式

【秘密指令】
    输入 kkkk   开启/关闭无限子弹

================================================================================
                              游戏系统
================================================================================

【玩家属性】
    生命值：    100 点
    体力值：    100 点
    初始子弹：  0 发
    最大子弹：  50 发

【武器系统】
    ┌─────────────────────────────────────────────────────────┐
    │ 武器      │ 类型      │ 伤害    │ 冷却时间  │ 射程      │
    ├─────────────────────────────────────────────────────────┤
    │ 剑        │ 近战      │ 20      │ 0.5秒     │ 7米/60°  │
    │ 手枪      │ 远程      │ 25      │ 0.3秒     │ 300米    │
    └─────────────────────────────────────────────────────────┘

【体力系统】
    - 加速奔跑时体力以 30点/秒 消耗
    - 站立时体力以 15点/秒 恢复
    - 行走时体力以 7.5点/秒 恢复
    - 体力耗尽后需恢复至 10% 才能再次加速

【敌人 AI】
    敌人拥有以下状态机行为：
    - 空闲(IDLE)     ：漫无目的等待
    - 巡逻(PATROL)   ：在区域内随机移动
    - 追击(CHASE)    ：发现玩家后追向玩家
    - 攻击(ATTACK)  ：接近玩家后进行攻击
    - 逃跑(FLEE)     ：血量低时逃离玩家
    - 死亡(DEAD)     ：被击杀后的状态

    敌人难度递进：
    - 每 30 秒难度提升 1 级
    - 每级属性提升 20%（血量、伤害、速度）
    - 敌人体型变大
    - 敌人大厅数量增加

【Boss 战斗】
    - 每 10 秒生成一个 Boss
    - Boss 拥有 500 生命值，30 伤害
    - Boss 体型是普通敌人的 3 倍

    Boss 技能：
    - 冲锋(CHARGE)  ：8秒冷却，快速冲向玩家
    - 砸地(SMASH)   ：12秒冷却，范围伤害+屏幕震动
    - 召唤(SUMMON)  ：15秒冷却，召唤小怪协助战斗
    - 狂暴模式      ：30%血量时触发，全属性大幅提升

【击败奖励】
    击杀普通敌人：
    - 回复 15 点体力
    - 掉落 1 发子弹（击败 Boss 后掉落增加）

    击杀 Boss：
    - 玩家基础速度 +2
    - 玩家冲刺速度 +3
    - 子弹掉落数量 +1
    - 显示"能力增强"提示

【小地图】
    - 显示范围：200 x 200 米
    - 绿色标记：玩家位置
    - 红色小点：敌人位置
    - 紫色大点：Boss 位置
    - 绿色扇形：玩家视野方向
    - 按 M 键可切换镜像模式

================================================================================
                              胜利条件
================================================================================

    游戏没有固定终点，目标是：
    1. 尽可能长时间生存
    2. 击杀更多敌人和 Boss
    3. 不断提升自己的能力

    当玩家生命值归零时游戏结束，显示存活时间和击杀统计。
